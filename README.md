### **Prerequisites**
- Python 3.8 or higher

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/jjingofarouk/TubeSnap.git
   cd TubeSnap
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt

   ```
   http://127.0.0.1:5000
